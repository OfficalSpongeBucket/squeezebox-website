SqueezeBox Documentation

Overview

SqueezeBox is a hybrid text + folder compression utility designed to make file shrinking fast, simple, and automatic. Version 2.13.8 introduces a redesigned UI and full‑disk scanning for files and folders.

Main Menu

When you launch SqueezeBox, you’ll see the main menu where you can choose to compress, decompress, or exit.

Compression Mode

Enter the name of a file or folder. SqueezeBox will automatically search your entire system — including external drives.

File Selection

If multiple matches are found, SqueezeBox will list them and let you choose the correct one.

Decompression Mode

To decompress, enter the name of a .sqz file. SqueezeBox will locate it and extract the contents.
